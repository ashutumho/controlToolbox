Control Toolbox

  • Control Toolbox
    • chgFreqUnit — Convert frequency unit of the frequency domain data
    • chgTimeUnit — Convert time unit of the system
    • dsort — sort the discrete domain pole by magnitude
    • esort — sort the continuous domain pole by real part
    • extractpid — Extract the parallel PID parameters from the transfer function
    • extractstdpid — Extract the standard PID parameters from the transfer function
    • findct — It finds that the given transfer function is of continuous domain type.
    • finddt — It finds that the given transfer function is of continuous domain type.
    • isEmpty — Determines that given state space, transfer function, matrix, array and hyper-matrix is empty
    • isfinite — Checks whether the system has finite coefficients
    • isproper — finds that the system is proper
    • order — finds model order
    • pid — Generate parallel form of PID controller, verify the parallel form of PID controller
    • stdpid — Generate standard form of PID controller, verify the standard form of PID controller
